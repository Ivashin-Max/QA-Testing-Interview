# QA-Testing-Interview

Репозиторий содержит тестовое задание для оценки QA-кандидатов на собеседовании.

## Что внутри

Статическая HTML-страница - форма регистрации вымышленного сервиса **SkyNova** с намеренно внедрёнными дефектами разного уровня критичности.

**Живая ссылка:** https://ivashin-max.github.io/QA-Testing-Interview/

## Как использовать

1. Отправить кандидату живую ссылку или открыть её прямо на собеседовании.
2. Дать устную вводную: кандидат видит форму регистрации - задача найти баги и оформить баг-репорты.
3. Оценить: полноту покрытия, структурированность подхода, качество описания дефектов, корректность severity.

## Обновление содержимого

Файл `index.html` - единственный исходник. После правок:

```bash
git add index.html
git push
```

`git push` синхронно отправляет изменения на GitHub (пересобирает Pages) и на Bitbucket.
